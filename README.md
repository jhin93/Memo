깃 원격 브랜치 가져오기  
https://velog.io/@juno7803/git-%EC%9B%90%EA%B2%A9-%EB%B8%8C%EB%9E%9C%EC%B9%98-%EA%B0%80%EC%A0%B8%EC%98%A4%EA%B8%B0remote-branch  

  로컬에서 원격 브랜치 상황 동기화하기
  $ git remote update

  원격 브랜치 목록 조회.
  $ git branch -r

  원격 브랜치와 동일한 내용으로 브랜치 생성하기.
  $ git checkout -b [생성할 브랜치 명] [원격 브랜치 명]
